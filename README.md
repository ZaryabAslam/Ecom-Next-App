# my-ecom-admin (Next.js 14.2.5 TypeScript starter)

## Quick start

1. Extract the archive.
2. Create `.env.local` with `MONGODB_URI` and `JWT_SECRET`.
3. Run:
   ```
   yarn install
   yarn dev
   ```
4. Visit http://localhost:3000

This starter includes:
- Next.js 14 (App Router)
- TypeScript
- Mongoose + a sample db connector
- Zod validator example
- A sample API route to test DB
